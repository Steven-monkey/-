# 						前后端交互

- #### Node.js模块化开发

  - ##### 模块内部可以使用`exports`对象进行成员导出，使用`require`方法导入其他模块

  ```javascript
  //文件1
  const sum = (num1, num2) => { return num1 + num2 }
  exports.sum = sum
  //文件2
  var add = require("./02 node模块化开发")
  console.log(add.sum(10, 20));
  ```

  - ##### 当模块导出并指向不同地址时，以module.exprot={}为准
